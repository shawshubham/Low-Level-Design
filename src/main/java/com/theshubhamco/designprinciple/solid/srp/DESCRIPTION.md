## Single Responsibility Principle (SRP) – Java Example

---

This module demonstrates the Single Responsibility Principle (SRP), one of the core SOLID design principles, through a real-world use case: an Employee Management System.

### ✅ What You’ll Learn
- What SRP means in practice
- How to identify SRP violations in code
- How to refactor large classes into smaller, focused ones
- How SRP improves modularity, maintainability, and testability
